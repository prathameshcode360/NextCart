# 🛒 NextCart - E-Commerce Web Application

NextCart is a modern and responsive **E-Commerce Web Application** built with **React**, **Redux Toolkit**, and **Firebase**. The application provides a seamless shopping experience where users can browse products, search items, manage their wishlist and cart, authenticate securely, and place orders.

---

## 🚀 Live Features

- 🛍️ Browse all available products
- 🔍 Search products instantly
- ❤️ Add/Remove products from Wishlist
- 🛒 Add products to Cart
- ➕ Increase/Decrease product quantity
- 🗑️ Remove products from Cart
- 💳 Place Orders
- 📦 View Order History
- 🔐 User Authentication (Login & Register)
- ☁️ Firebase Authentication & Firestore Database
- ⚡ Global State Management using Redux Toolkit
- 📱 Fully Responsive Design
- 🎨 Clean and Modern UI

# ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/nextcart.git
```

### Navigate to project folder

```bash
cd nextcart
```

### Install dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
